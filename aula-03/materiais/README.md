## Dependências locais incluídas

- `mapa-skills-artifacts.js`: catálogo dos artefatos e previews usados pelo mapa;
- `assets/al/`: identidade visual e ícones locais;
- `mapa-skills-samples/`: amostras navegáveis carregadas pelos previews;
- `data/project-brief.schema.json`: schema usado pelo briefing;
- `data/skill-unlock-rules.json`: regras usadas pelo briefing.

As bibliotecas Iconoir, Marked, PDF.js, Mermaid, Tailwind CDN e Font Awesome continuam sendo carregadas por CDN, conforme os arquivos originais. O detalhamento está em [`../docs/mapa-skills-dependencias.md`](../docs/mapa-skills-dependencias.md).

Os dois manuais foram copiados das versões revisadas existentes fora deste pacote. O material dentro de `aula-03/materiais/` é a cópia didática versionável da Aula 3.
