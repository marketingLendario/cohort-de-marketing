# Exemplo público de três semanas

Este diretório contém somente entradas sintéticas minimizadas:

- `weekly-panels.jsonl`: três `WeeklyPanelV1`, um por semana;
- `source-observations.json`: observações confirmadas e divergentes de plataforma, checkout e caixa;
- `previous-decision.json`: decisão humana estruturada que será confrontada com o resultado;
- `expected-walkthrough.json`: resumo determinístico esperado.

Não edite os arquivos durante o walkthrough. Escolha um diretório de saída vazio
e siga a sequência em [../../README.md](../../README.md).
