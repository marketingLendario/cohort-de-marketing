# Third-party notices and redistribution gate

Status: **CLEARED FOR DISTRIBUTION** with the license boundaries below.

The public release manifest is regenerated by
`scripts/generate-ads-creative-factory-manifest.mjs` at the repository root.
It hashes only the exact allow-list recorded in `source-manifest.json`; it does
not include client packs, people, photos, logos, generated references, campaign
examples or secrets. The manifest records provenance, licensing and the rights
verdict for every released file.

Client identity and people assets remain in external packs. Those packs are
not a grant of redistribution rights. Local execution must opt into one
explicitly, for example:

```bash
ACF_BRAND_PACK="$PWD/client-packs/example/pack.json" \
  python3 .claude/skills/ads-creative-factory/scripts/factory.py campaign.yaml
```
