# Pasta de GUIAS — "Estou perdido em X. O que fazer?"

> A regra desta pasta: **um problema → um guia → o próximo passo.**
> Ache seu problema na tabela, abra o guia e siga. Cada guia mostra o caminho SIMPLES funcionando — você adapta ao seu contexto depois.
> Convenção: `.md` = pra IA ler e editar · `.html` (quando existir) = pra você ler no navegador. Mesmo conteúdo.
